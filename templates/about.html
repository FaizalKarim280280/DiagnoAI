About

Disease NLP is a tool to detect a disease from a text description of the patient's symptoms and daily condition. It is based on a transformer model called BERT[1], fine-tuned for 24 common diseases.

Motivation

As healthcare systems worldwide continue to face challenges posed by the COVID-19 pandemic, there is an increasing need for innovative solutions to help individuals receive medical attention and care remotely. One such solution is a tool that can detect a person's disease based on a brief textual description of their symptoms and daily condition.

People living in rural and under served areas may face barriers to visiting a doctor, such as transportation or COVID-19 restrictions. A tool that can detect diseases based on a description of symptoms could be a valuable resource for these individuals. It could allow them to receive a diagnosis remotely and potentially connect them with the appropriate treatment options.

A tool that can detect diseases based on a description of symptoms could potentially reduce the number of in-person appointments needed, freeing up resources and easing the burden on healthcare systems. With the rise of tele medicine and remote healthcare options, it is becoming increasingly common for individuals to receive medical attention and care remotely.

Novelty

There are limited tools on the internet that solves the task of disease diagnosis. This tool is different from the others because the user can express themself in natural language, thus enabling the user to describe the symptoms in detail. Most of the existing tools either rely on reserved keywords for the symptoms or a long list of questions which might be irrelevant to the user.

There can be scenarios where a user might want to describe a symptom in a particular way which might not be present in the list of reserved keywords. Using a language model solves this problem as the model can be trained to understand the relationship between words, understand the context and finally, make the predictions.

DiseaseNLP is a machine learning tool that was developed to assist in the diagnosis of various medical conditions. It uses advanced algorithms to analyze patient symptoms and provide estimates of possible diagnoses.
